# Incapacitação do Piloto

5.6. Incapacitação do Piloto
Em caso de incapacitação do piloto (Pilot Incapacitation),o piloto remanescente deve chamar o CSB/CF o mais rápido possível.
Procedimento de Pilot Incapacitation O CSB/CF deve entrar no cockpit imediatamente, e:
- Tirar os pés do piloto do pedal;
- Colocar o assento totalmente para trás;
- Reclinar o encosto do assento;
- Atar e travar o cinto de inércia do piloto incapacitado;
- Solicitar assistência médica;
- Verificar se existe algum tripulante extra a bordo,
capacitado para assumir o lugar do piloto.
Para o peso de um corpo inconsciente, duas pessoas deverão atuar. O trabalho em equipe neste momento é fundamental.
5.6.1.
Acesso ao Cockpit com Pilot Incapacitation Se não for possível contato via interfone com o cockpit (pilotos incapacitados), o acesso deve ser feito digitando o código de acesso de emergência - Emergency Access Code.
