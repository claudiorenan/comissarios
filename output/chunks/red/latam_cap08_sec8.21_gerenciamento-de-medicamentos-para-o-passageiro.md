# Gerenciamento de Medicamentos para o Passageiro

8.21. Gerenciamento de Medicamentos para o Passageiro
A tripulação de cabine não deve disponibilizar medicamentos a clientes ou aplicar injeções, tampouco a um menor desacompanhado. Quando um passageiro informar a um tripulante de cabine que tem algum sintoma ou solicitar um medicamento, deve-se perguntar sobre seu estado de saúde e se gostaria de receber atendimento médico.
Não é permitido armazenar medicamentos de clientes. Se um cliente levar consigo um medicamento que requeira refrigeração, o tripulante pode oferecer um saco de enjoo para proteção deste medicamento, e um saco adicional com gelo para que ele insira o medicamento já protegido, advertindo-o de que mesmo assim o medicamento poderá sofrer danos. O medicamento deve ser mantido sob os cuidados do cliente durante todo o voo.
Como cordialidade, recomendamos conferir periodicamente a condição do gelo, para possível reposição.
