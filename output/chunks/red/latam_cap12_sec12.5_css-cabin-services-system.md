# (CSS) Cabin Services System

12.5. (CSS) Cabin Services System
É um sistema computadorizado desenvolvido para gerenciar o status da cabine. Ele pode controlar e monitorar as funções da cabine de passageiros por meio de telas touchscreen denominados CAP (Cabin Attendant Panel).
