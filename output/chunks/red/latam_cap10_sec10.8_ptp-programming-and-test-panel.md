# (PTP) Programming and Test Panel

10.8. (PTP) Programming and Test Panel
Os anúncios de pane na cabine de passageiros são dados no painel de teste e programação. O PTP é utilizado para programar e testar as indicações dos sistemas da cabine. Este sistema está presente apenas em aeronaves com FAP analógico.
O PTP está instalado ao lado do FAP e é acessado através de uma tampa plástica.

O painel PTP permite uma averiguação da condição dos sistemas, teste dos sistemas da cabine e programação da configuração.
Deve ser utilizado apenas pela manutenção. Nas aeronaves que possuem FAP digital, este painel é integrado ao próprio FAP.
