# Fornos B/E Aerospace

10.13. Fornos B/E Aerospace
Os fornos B/E AEROSPACE aquecem as refeições utilizando um sistema de injeção de água. Para isso, a porta do forno fica completamente vedada, mantendo 100% da umidade interna.
A trava da porta possui três posições: open /safe /lock.
Para que o forno funcione a porta deverá estar sempre na posição lock.

Controle e indicações:

a) Display: O display indica o tempo/ tipos de aquecimento e mensagens de erro em caso de qualquer falha no sistema;
b) Teclado: O forno pode ser programado rápida e facilmente. O teclado consiste em seis funções e 4 indicações luminosas tendo, cada indicação luminosa, uma função específica;
c) ON/OFF: Liga e desliga o forno;
d) CLEAR/STOP: Para ou reseta um programa a qualquer momento;
e) START/PAUSE: Confirma um programa selecionado e inicia processo de aquecimento pausa do programa solicitado;
f) SETA PARA CIMA: Seleciona tempo para mais alterando o tempo de aquecimento para mais;

g) SETA PARA BAIXO: Seleciona tempo para menos alterando o tempo de aquecimento para menos;
